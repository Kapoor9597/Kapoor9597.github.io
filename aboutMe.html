Storage is fun – people always need more of it. It’s a bit difficult to estimate, but there was something like 4 zettabytes of storage on the planet, and that was 3 years ago. A zettabyte is 1,000,000,000,000,000,000,000 bytes or one billion 1TB drives. That’s a lot of disk drives!! And it’s only going to get bigger. So looking at all that data I started thinking about what you could do if you could analyze it. Not bits of it, but all of it. If you start looking at “the big picture” with data you can start to see some really interesting things.

This is Data Science. You take the data and look for the patterns and meaning behind the data. It’s a progression, from the data you get information, from the information you get knowledge, and from knowledge, wisdom.

And here I'm an aspiring data scientist with a big interest in how data shapes our lives, a Front-End Developer and Machine Learning Practitioner from India. I mainly work with Java for developing software applications and with Python for competing in Kaggle.


I read online tutorials (such as MachineLearningMastery!), watch videos (SentDex!), work with people, etc. to learn what I need to know to address the problem at hand. If you are curious, willing to fail over and over while learning, and interested in how to harness and make sense of data from all kinds of sources (including social media), you can become a data scientist.

In my spare time, the web development community is a big part of my life. Currently, I'm teaching myself Node JS, MongoDB and RESTful API for Back-End Development.
